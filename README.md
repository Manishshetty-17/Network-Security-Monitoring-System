# 🔐 Network Security & Monitoring System

A shell-based network monitoring and intrusion detection system that analyzes system activity and detects suspicious behavior.

---

## 📌 Overview
This project is developed using Bash scripting on Ubuntu. It helps in monitoring network activity, identifying suspicious IP addresses, and detecting possible security threats in real-time.

---

## 🚀 Features
- 🌐 Internet connectivity check  
- 🌍 Website status monitoring (HTTP response codes)  
- 🔗 Active network connections display (IP + Port)  
- 🔑 Failed login attempt detection  
- ⚠ Suspicious IP detection using log analysis  
- 📊 Real-time monitoring system  

---

## 🛠 Technologies Used
- Bash Shell Scripting  
- Linux Commands (`ss`, `netstat`, `curl`, `grep`, `awk`)  
- Ubuntu (WSL environment)  

---
